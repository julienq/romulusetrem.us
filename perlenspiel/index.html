Perlenspiel

Some experiments made with Brian Moriarty's Perlenspiel. Works best in Firefox.

  • Choooose: a tiny "choose your own adventure" engine with a short sample game.
  • Freeeeze: a small puzzle game set on an icy pond. Only a few levels so far, more to come. (Game B)
  • Picopolis: this will remind you of a very famous game... Choose tools with the following keys: R for residential, C for commercial, and I for industrial zoning; S for streets. The interface is working but not the simulation, so stay tuned for updates.
  • Mosaic: drag an image to make a mosaic.
  • Xylophone: a simple sequencer; toggle cells to play or mute notes; use up/down arrow key to change the tempo.
  • Colors: a simple color toy; move the mouse and the scroll wheel.
  • RGB Life: the obligatory game of life; this one being played on three colored planes at once. Click anywhere to shake things up a little.
